This is a full-stack web application for managing company data such as employees, job positions, departments, absences, and system users.

The application is intended for small and medium-sized companies and includes role-based access control.

Note: The application user interface is in Serbian, while this documentation is in English.

Technologies:
- Frontend: Vue.js 3, Vue Router, Axios, HTML, CSS
- Backend: Node.js, Express.js
- Database: MySQL

Main Features:
- Employee management
- Department and job position management
- Absence tracking
- User authentication and authorization
- Administrator and standard user roles

Running the project locally:
1. Import the database (if available)
2. Start backend using npm
3. Start frontend using npm
